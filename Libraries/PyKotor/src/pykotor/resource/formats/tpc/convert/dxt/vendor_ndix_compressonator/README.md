# ndixUR Compressonator (vendored)

`compressonator.js` is copied from [ndixUR/tga2tpc](https://github.com/ndixUR/tga2tpc) (GPL-3.0 application; this file carries the AMD MIT header from the original Compressonator C++ port).

`ndix_compress_cli.cjs` is PyKotor glue (MIT, same as PyKotor library) to invoke the same DXT1/DXT5 path as the Electron **tga2tpc** default profile (Compressonator, `RefinementSteps: 2`, normal speed).
