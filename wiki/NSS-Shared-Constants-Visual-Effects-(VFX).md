# Visual Effects (VFX)

Part of the [NSS File Format Documentation](NSS-File-Format).

**Category:** Shared Constants (K1 & TSL)

Visual effect **constants** used by NWScript and the [effects system](NSS-Shared-Functions-Effects-System) align with engine data in **[visualeffects.2da](2DA-File-Format#visualeffects2da)** (row labels, VFX types, and related columns). The master routine and constant index lives in [NSS-File-Format](NSS-File-Format) under **Shared Constants** and **TSL-Only Constants** for any game-specific additions.

- **2DA:** [2DA-visualeffects](2DA-File-Format#visualeffects2da) — table semantics and column overview
- **Scripted effects:** [NSS-Shared-Functions-Effects-System](NSS-Shared-Functions-Effects-System) — `EffectVisualEffect` and related calls
- **GFF:** [GFF-File-Format](GFF-File-Format) — VisualType and template fields where applicable

**TSL-only** index page (stable URL): [NSS-TSL-Only-Constants-Visual-Effects-(VFX)](NSS-TSL-Only-Constants-Visual-Effects-%28VFX%29).
