# BWM File Format

The authoritative BWM (Binary Walkmesh) specification for the Odyssey engine lives in **[Level Layout Formats — BWM section](Level-Layout-Formats#bwm)**.

Normative format text is grounded in reverse engineering and community pipelines (KOTORMax, kotorblender, kaurora). The wiki may include a **non-normative** PyKotor implementation subsection when explicitly labeled; see [authoritative BWM policy](../docs/solutions/documentation/authoritative-bwm-wiki-from-re-and-pipelines.md).
